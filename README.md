# DSA Assignment Solutions

This repository contains solutions to the following programming questions:

1. **Days to Seconds**
   - Asks the user for the number of days and prints the number of seconds in that many days.
2. **Volume of a Sphere**
   - Asks the user for the radius and calculates the volume of a sphere using the built-in exponential function.
3. **Area and Perimeter of a Square (Using Functions)**
   - Asks the user for the side length of a square and displays its area and perimeter.
4. **Character Case Checker (Using Functions)**
   - Determines whether a character input by the user is uppercase or lowercase.
5. **Pseudocode Conversion**
   - Implements a loop that updates variables x and y as described in the provided pseudocode, then displays x.
6. **Array Average Calculator**
   - Uses a loop to input 5 values into an array and calculates/displays their average.
7. **Codility Data Structures Challenge**
   - Attempted on Codility; see the included screenshot (`question7-codility test.png`) for the solution and analysis.

---

All files compile and run successfully. Each question is implemented in its respective C++ file (question1.cpp to question6.cpp). For question 7, refer to the attached image.
